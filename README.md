# 花影

尝试做一个在浏览器中运行的录屏工具

![wireframe](./doc-assets/wireframe.png)

**功能**  
- [ ] 录制功能
  - [x] 录制视频
  - [x] 暂停、恢复录制
  - [ ] 录制音频
  - [ ] slider 起止时间点预览
- [ ] 素材管理
  - [x] 捕获屏幕、演讲者（摄像头+麦克风）
  - [x] 增删素材
  - [x] 素材层级拖拽排序
  - [ ] 裁剪屏幕素材
- [ ] 导出资源
  - [x] 导出视频（mp4）
  - [x] 资源预览
  - [x] 资源删除
  - [ ] 导出动图
- [ ] UI 优化
- [ ] 多素材存储为模板
- [ ] 部署到 Github Pages
